9. java环境配置与安卓病毒监听

亙壬2023-12-042023-12-06

java环境配置

AhMyth是一款能控制安卓手机的远程控制软件

理论上需要java环境配置

项目地址是在GitHub - AhMyth/AhMyth-Android-RAT: Android Remote Administration Tool

下载Java8 的配置文件

安装

双击下载下来的jdk

然后下一步，注意，这里我们用的是默认路径

后面会提醒在安装一遍，选择一个空文件夹

到此，安装完成 测试一下

Windows键+R搜索cmd回车打开终端

然后输入java -version回车

注意，在环境配置中

一个是用户变量；一个是系统变量

用户变量，就是在用户变量里面设置的值，只对当前windows账号起作用

系统变量中所设置的值，会针对这一台电脑中所有的用户都起作用

安装mumu模拟器

官网下载mumu模拟器安装

选择mumu模拟器6 x64位下载安装

等待安装完成

查看物理机ip

ip为192.168.2.50

Source IP 我们配置自己的服务器ip

Source Port 是我们上一个界面配置的监听端口（我这边默认配置的是42474）

点击Build生成木马

成功生成 此木马路径在C:\Users\admin\AhMyth\Output\Ahmyth.s.apk

右键使用mumu打开安装

点击一下即可上线

这里是我们配置监听端口的界面

默认是42474 只要端口不冲突即可

然后点击Listen

功能依次是，拍照 ，控制文件，录音，查看手机所在地图，查看通迅录，发短信，查看通话记录。

Camera 照相机 允许受感染的Android设备使用摄像头拍照

File Manager 文件管理器 文件管理器允许从受感染的Android设备中访问文件

Mic 麦克风 允许从受感染的Android设备录音

Location 位置 显示地理位置模块和目标Android设备的位置

contacts 触体 允许从受感染的Android设备访问通讯录

SMS 信息 可以将SMS消息从受感染的Android设备发送到其他移动设备，还可以从目标Android设备查看SMS列表

Calls Logs 通话记录 允许从受感染的Android设备访问通话记录
